MVP: Replaces WooCommerce My Account orders with Zeron obligations view (Dummy/Live mode).

== Description ==
This plugin adds a new My Account tab "Задължения" with unpaid/paid documents.
Dummy mode uses bundled JSON. Live mode is a placeholder (settings only).

== Installation ==
1. Upload folder to wp-content/plugins/
2. Activate
3. Admin → Global Zeron orders → set Dummy
